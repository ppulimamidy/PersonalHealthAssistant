## üöÄ Usage Examples

### Basic Condition Detection
```bash
curl -X POST "http://localhost:8008/api/v1/health-analysis/analyze" \
  -F "image=@skin_rash.jpg" \
  -F "body_part=skin" \
  -F "symptoms=redness,itching" \
  -F "urgency_level=normal"
```

### Get Supported Conditions
```bash
curl "http://localhost:8008/api/v1/health-analysis/supported-conditions"
```

### Medical Query Processing
```bash
curl -X POST "http://localhost:8008/api/v1/health-analysis/medical-query" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "What are the symptoms of diabetes?",
    "query_type": "symptom",
    "urgency_level": "normal"
  }'
```
